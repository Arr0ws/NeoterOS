# Neoter OS 🖥️

A fake terminal-based OS made with Javascript (Node.js)

---

## 📦 Requirements
- [Node.js](https://nodejs.org/)
- npm (comes with Node.js) or pnpm/yarn

---

## 🔧 Installation
Clone this repo and install dependencies:

```bash
git clone https://github.com/Arr0ws/NeoterOS.git
cd NeoterOS
npm install
